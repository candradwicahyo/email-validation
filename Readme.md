# Email Validation

sebuah program sederhana untuk memvalidasi sebuah input, apakah value dari input tersebut berisi format email atau tidak.

### dibuat dengan berikut **:**

* HTML
* CSS
* JavaScript

### lainnya

* font (Roboto)

### alat yang digunakan **:**

* handphone
* code editor (Acode)
* browser (Google Chrome)
